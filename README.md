# üåç GlobEx: Global Economy Bot Template

GlobEx is a clean, modular, and easy-to-use Discord economy bot template built with `discord.py` and `SQLite`. It features a **Global Economy** system, meaning user balances persist across all servers the bot is in.

## ‚ú® Features
* **Global Balances:** SQLite database keeps track of Wallet and Bank balances.
* **Cogs Architecture:** Easy to add new commands in organized modules.
* **Economy Commands:** `balance`, `beg`, `work`, `pay`, and `slots`.
* **Leaderboard:** A global top 10 wealth ranking system.
* **Cooldowns:** Built-in rate limiting to prevent spam.
